# aspish Demo

A demonstration of the `aspish` Python package running within a Cloudflare Worker.

## Usage
Provides an interactive frontend to input family parent relationships and uses `aspish` (ASP - Answer Set Programming) to compute siblings.

## Deployment
1. Run `npm install`
2. Run `npm run deploy` to deploy to Cloudflare Workers.
