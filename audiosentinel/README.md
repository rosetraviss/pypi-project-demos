# audiosentinel Demo Worker

This is a Cloudflare Worker demo for the [`audiosentinel`](https://pypi.org/project/audiosentinel/) PyPI package.
It provides an interface to detect whether an uploaded audio file is human-recorded or AI-generated.

## Local Development

Ensure you have `uv` and Node.js installed.

1. Navigate to this directory:
   ```bash
   cd audiosentinel
   ```
2. Start the local development server:
   ```bash
   npx wrangler dev
   ```
